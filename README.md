# Prediction-of-CO-solubility-in-Ionic-liquids
CO₂ emissions are a big threat to the environment and has to be controlled. It is found that CO₂ has good solubility in ionic liquids. Solubility depends on various features of the ionic liquid, temperature etc. Experimenting on every liquid is inefficient. Developed machine learning models to predict the solubility of CO₂ in Ionic liquids.
